INFORMATION

Ember Commands

Basic Commands:
`ember new app-name` - sets up a new project with all the necessary files and folders
`ember serve` - Runs the app locally at http://localhost:4200
`ember build --environment=production` - build for production

Generate new pieces for the app:
Component: `ember generate component component-name`
Route: `ember generate route route-name`
Model: `ember generate model model-name`

Folder Structure:
app/: Main codebase (components, routes, templates, etc.).
tests/: Automated test files.
public/: Static assets (images, etc.).
config/: Configuration files.
